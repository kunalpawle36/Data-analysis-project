# Classic Business Operations Analysis

This project analyzes business operations of a classic model car company using the classicmodels database.

## Overview
- Dataset: MySQL sample database with tables for customers, employees, products, orders, etc.
- Focus: Sales performance, customer insights, operational efficiency.

## Key Components
- **Raw Data**: Original database dump.
- **Clean Data**: Processed datasets after SQL cleaning.
- **EDA**: Exploratory analysis scripts.
- **SQL & Data Cleaning**: Queries, schema definitions, and views for data preparation.

## SQL Views and Data Schema
- **Data Schema**: Defines table structures, relationships, and constraints for data integrity.
- **SQL Views**: Created for aggregated views like customer summaries, sales reports, and product performance without altering raw data.

## Methodology
- Data cleaning with SQL (handling missing values, duplicates).
- Schema review and adjustments.
- View creation for analysis and reporting.

## Tools
- MySQL for queries and views.
- Python for EDA.
- Visualization tools for insights.

## Results
- Insights into trends, top performers, and recommendations.
