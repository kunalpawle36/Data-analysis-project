# Raw Data

This folder contains the raw data for the Classic Business Operations Analysis project.
The primary dataset is the classicmodels database dump (`dummy.sql`), which includes tables for customers, employees, products, orders, etc.
The data is sourced from the MySQL sample database and represents a model car business.
Use this data for exploratory data analysis, SQL queries, and deriving business insights.
For cleaned or processed data, refer to the Clean_data folder.
